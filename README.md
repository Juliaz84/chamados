# Sistema de Chamados 

Sistema de gerenciamento de chamados desenvolvido em **Python** usando **Tkinter** e **SQLite**.  
Permite cadastrar, consultar, atualizar e excluir chamados de clientes, com controle de prioridade e status.

---

## Funcionalidades

- Cadastrar novos chamados com:
  - Cliente
  - Problema
  - Prioridade (Baixa, Média, Alta, Crítica)
  - Status (Aberto, Em andamento, Aguardando, Fechado)
  - Técnico responsável
- Listar chamados em uma tabela interativa
- Editar chamados existentes
- Excluir chamados
- Resetar IDs e limpar a tabela
- Interface gráfica moderna e intuitiva

---

## Pré-requisitos

- Python 3.10 ou superior
- Bibliotecas:
  - tkinter (já vem com Python)
  - sqlite3 (já vem com Python)
